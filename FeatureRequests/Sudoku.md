A timed Sudoku with a daily, seasonal, and all-time leaderboard.

Eventually we will host many variants of Sudoku, but we need to implement a basic board first.  Each board should have a hash and a leaderboard for that hash.
The leaderboard should have a clickable hash of the game that was played.  If you go to the daily link everyone gets the same board, but there should also be a new game
option where you get a random board.  All games played that day go on the high scores list.

The games should be Proper Sudoku[4] – A Sudoku with one solution. Sudokus are always expected to be proper Sudokus unless the puzzle setter specifies otherwise.

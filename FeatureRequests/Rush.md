Key Rules of MineSweeper Rush
Gameplay Mechanics: The board constantly grows upward, requiring fast, continuous clearing.
Hitting a Mine: Unlike classic Minesweeper, clicking a mine does not end the game; instead, two new rows are immediately added.
Winning/Clearing Rows: A row is cleared when all mines in it are marked.
Safety Net: The game features an AI system that ensures no penalty if no safe move exists.
Progression: If fewer than four rows remain, four new rows appear, and the game speed increases with the score.
Modes: Players can choose between Easy, Normal, and Hard modes, or customize board size and mine count.
Number of Mines Found, Score and Time are saved on the server with a separate Leaderboard for the Rush variant.
As the timer ticks, a row at the top of the board fills with boxes (with an x). When the row fills, the boxes got from x to blank and that row becomes playable.

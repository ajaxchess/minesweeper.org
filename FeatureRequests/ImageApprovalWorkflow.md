15-puzzle has a puzzle generator https://minesweeper.org/other/15puzzle/generator
that allows uploading images.
There is an admin page for approving the images
https://minesweeper.org/admin/15puzzle-photos
and deleting the images that violate our policies.

jigsaw has a puzzle generator https://minesweeper.org/other/jigsaw/generator
that allows uploading images.


We want a workflow where images are not able to be displayed until they are
approved on the admin page.
The one exception is that if you upload the image and you are logged in,
you should be able to enter the game URL from the generator.  The game will
be displayed, but a note at the top will let the logged in user know that
the image is being evaluated by the website admins and will be processed shortly.

The admin pages for 15-puzzle should have a group of puzzles that have not been
evaluated at the top of the page.  They should all have a Delete button and
there should be an Approve button.
Lower down on the page should be a list of approved puzzles.
There should be a Delete button beside each of those images in case the 
admin staff changes their mind about an image.

We want to make a Mahjong Solitaire with the ability to guarantee a win for
any served board.
The tiles should be arranged in the Turtle formation.
There should be a button to show tiles that can be moved.
The goal is to match 2 tiles and the found tiles should be displayed below 
the playing field.
The games will be timed and there will be a daily puzzle.  
Each board will have a hash and a high scores table for that board.
There should be a daily, seasonal, and all time leaderboard.
There should be an undo move button.
